# Contexto
A alfabetização na infancia é um dos pilares para o dessenvolvimento educaional, social e economico de um pais.

O Compromisso Nacional Criança Alfabetizada visa garantir que todas as crianças estajam alfabetizadas até o final do ***2° ano do ensino fundamental***. 

Para apoiar este objeivo o Instituto Nacional de Estudos e Pesquisas Educacionais Anísio Teixeira em 2023 realizou a pesquisa Alfabeiza Brasil onde foi definido o parâmetro ***ponto de corte de 743 pontos na escala de proficiência do Saeb***, nivel onde uma criança pode ser considerada alfabetizada.

Com base neste parâmetro foi criado o ***Indicador Criança Alfabetizada***, percentual de estudantes que atingem esta proficiência.

A meta é: ***até 2030, todas as crianças brasileiras estejam alfabetizadas ao final do 2º ano do ensino fundamental***

## Objetivo
Compreender os fatores que influenciam o processo de algabetização integrando diferentes fontes de dados como: Metas Nacionais E Estaduais, Metas Municipais, Dados Territoriais, Microdados Educacionais e Indicadores De Desempenho.

Fonte de dados: Indicador Criança Alfabetizada - Base dos Dados (https://basedosdados.org/dataset/073a39d4-89cf-4068-b1e8-34ed0d9c0b72?table=e1de7a6a-5038-4e81-89f0-a15f2cc12c9b)
